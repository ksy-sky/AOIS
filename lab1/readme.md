# Лабораторная работа №1
## Представление чисел и арифметика в компьютере(вариант е)


## Реализованные функции

### 1. Конвертация целых чисел (IntegerConverter)

| Функция | Описание |
|---------|----------|
| `to_direct(value)` | Прямой код (знак + модуль) |
| `to_inverse(value)` | Обратный код (инверсия для отрицательных) |
| `to_complement(value)` | Дополнительный код (обратный + 1) |
| `from_direct(bits)` | Из прямого в десятичное |
| `from_inverse(bits)` | Из обратного в десятичное |
| `from_complement(bits)` | Из дополнительного в десятичное |

**Особенности:**
- Диапазон: [-2³¹, 2³¹-1]
- Минимальное число -2³¹ обрабатывается отдельно
- Для отрицательных чисел: обратный = инверсия модуля, дополнительный = обратный + 1

---

### 2. Арифметика в дополнительном коде (ComplementArithmetic)

| Функция | Описание |
|---------|----------|
| `add(a, b)` | Сложение в доп. коде |
| `subtract(a, b)` | Вычитание: a - b = a + (-b) |
| `negate(bits)` | Отрицание (инверсия + 1) |

**Особенности:**
- Проверка переполнения (знаки слагаемых одинаковые, результат другой)
- Использует BitUtils для сложения

---

### 3. Арифметика в прямом коде (DirectArithmetic)

| Функция | Описание |
|---------|----------|
| `multiply(a, b)` | Умножение в прямом коде |
| `divide(a, b)` | Деление в формате 16.16 fixed-point |

**Особенности:**
- Умножение: поразрядное сложение со сдвигом
- Деление: целая часть + 16 бит дробной части (точность 5 знаков)
- Формат 16.16: 16 бит целая часть, 16 бит дробная

---

### 4. Конвертер IEEE-754 (Ieee754Converter)

| Функция | Описание |
|---------|----------|
| `to_ieee754(value)` | float → 32-битный IEEE-754 |
| `from_ieee754(bits)` | IEEE-754 → float |

**Поддержка спецзначений:**
- NaN (не число)
- ±Infinity (бесконечность)
- Денормализованные числа

---

### 5. Арифметика IEEE-754 (Ieee754Arithmetic)

| Функция | Описание |
|---------|----------|
| `add_dec(a, b)` | Сложение |
| `subtract_dec(a, b)` | Вычитание |
| `multiply_dec(a, b)` | Умножение |
| `divide_dec(a, b)` | Деление |

**Флаги операций:**
- `FLAG_OVERFLOW` - переполнение
- `FLAG_UNDERFLOW` - потеря точности
- `FLAG_INEXACT` - неточный результат
- `FLAG_INVALID` - некорректная операция

---

### 6. Gray BCD (GrayBCD)

| Функция | Описание |
|---------|----------|
| `to_gray_bcd(value)` | Десятичное → Gray BCD (8 цифр) |
| `from_gray_bcd(bits)` | Gray BCD → десятичное |
| `add(a, b)` | Сложение в Gray BCD |

**Таблица Gray BCD:**
| Цифра | Gray BCD |
|-------|----------|
| 0 | 0000 |
| 1 | 0001 |
| 2 | 0011 |
| 3 | 0010 |
| 4 | 0110 |
| 5 | 0111 |
| 6 | 0101 |
| 7 | 0100 |
| 8 | 1100 |
| 9 | 1101 |

**Особенности:**
- 8 десятичных цифр (0 - 99,999,999)
- Поразрядное сложение с переносом
- При переполнении результат обнуляется
